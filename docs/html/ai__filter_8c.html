ai_code_t* ai_filter ( FILE *  input)

Filter the code and return the code.

Parameters
inputThe file for read the instructions
Returns
The linked list to the code

Reads the instructions filtering all non-valid characters and removing the commentaries. Stops in the End Of File. The AI_INSTRUCTIONS macro is the list of valids characters. AI_COMMENTARY is the character to one-line commentaries.
